🔐 Customer Login Setup
Wire your website login/signup to Google Sheets
ℹ️
Your website at visual-exact-frame.lovable.app has Sign In / Sign Up buttons. Here's how to make them active and save customer data automatically.
Step 1: Go to your n8n dashboard
Step 2: Create a new workflow with a Webhook trigger
Step 3: Name it "Customer Auth Handler"
Step 4: Add Google Sheets node → Append row to "Customers" tab
Step 5: In your website code (Lovable), connect the Sign Up form to POST to this webhook
Step 6: Each signup automatically saves: Name, Email, Password Hash, Signup Date, Status

⚠️ Important: Never store plain text passwords. Use n8n's Code node to hash them first (e.g. SHA-256).
